# CS 370 Portfolio Reflection

## Project Summary

For Project Two, I worked on a pirate intelligent agent for a treasure hunt game. The goal of the project was to train a non-player character, or pirate agent, to navigate a maze and find the treasure. The project focused on reinforcement learning, deep Q-learning, neural networks, pathfinding, and intelligent agent design.

I was given starter code that included the maze environment, helper classes, and some of the structure needed to run the game. The provided code included files such as the `TreasureMaze` environment and the `GameExperience` class, which helped manage the agent’s interactions with the maze and store experience data. I did not modify the provided `.py` files. The code I created myself was mainly in the Q-training section of the Jupyter Notebook. I implemented the deep Q-learning training loop, where the pirate agent explored the maze, selected actions, received rewards or penalties, stored experiences, trained the neural network model, and improved its ability to reach the treasure.

## What Do Computer Scientists Do and Why Does It Matter?

Computer scientists solve problems by designing, building, testing, and improving software systems. They use logic, algorithms, data structures, programming, and analysis to create solutions that can help people, businesses, and society. Computer science matters because technology is used in almost every part of life, including communication, healthcare, transportation, cybersecurity, education, entertainment, and business operations.

This project showed me how computer scientists can use artificial intelligence to solve complex problems. Instead of manually programming every possible path through the maze, I used reinforcement learning so the agent could learn from experience. This connects to real-world applications such as robotics, navigation systems, game development, automation, and decision-making systems.

## How Do I Approach a Problem as a Computer Scientist?

As a computer scientist, I approach problems by first understanding the goal, then breaking the problem into smaller parts. For this project, the main goal was for the pirate agent to find the treasure. To solve that problem, I needed to understand the maze environment, the possible actions, the reward system, the neural network model, and the training process.

I also learned the importance of testing and debugging. During this project, I had to make sure the notebook cells were run in the correct order, imports were loaded, helper functions were defined, and the training loop worked without errors. This helped me understand that solving a programming problem is not only about writing code, but also about testing the code, reading errors carefully, and improving the solution until it works correctly.

## What Are My Ethical Responsibilities to the End User and the Organization?

My ethical responsibility to the end user is to design systems that are reliable, fair, safe, and transparent. Even though this project was a game, the same principles apply to larger AI systems. Users should be able to trust that an intelligent system is working as intended and not making harmful or unfair decisions. AI systems should also be tested carefully to make sure they perform correctly in different situations.

My ethical responsibility to the organization is to create solutions that meet the project goals while following best practices. This includes writing readable code, using comments, avoiding unnecessary changes to provided files, and testing the final product before submission. It also means being honest about what the system can and cannot do. In real-world AI development, organizations must also consider privacy, bias, security, and accountability when building intelligent systems.

## Course Reflection

This course helped me better understand artificial intelligence and how intelligent agents learn. I learned that AI is not just about creating systems that appear smart. It requires data, algorithms, training, evaluation, and ethical decision-making. Through this project, I learned how reinforcement learning allows an agent to improve through rewards and penalties. I also learned how deep Q-learning uses a neural network to estimate the best action to take in a given state.

Overall, this project helped me connect computer science theory to a practical application. The pirate intelligent agent showed how algorithms and neural networks can be used to solve a pathfinding problem. This experience strengthened my understanding of AI, problem-solving, debugging, and responsible software development.
